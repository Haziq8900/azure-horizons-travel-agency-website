# 🌍 Azure Horizon Travel Agency

**Journeys Beyond Imagination**

A responsive front-end website for a fictional travel agency, built with plain HTML and CSS. The site showcases travel destinations, curated vacation packages, and a booking flow for custom tours.

## ✨ Features

- **Destinations gallery** — showcases 12 destinations (Paris, Dubai, London, Sydney, Skardu, Istanbul, and more), each with an image, short description, and route code (e.g. `KHI → PAR`)
- **Packages table** — detailed vacation packages with duration, pricing, included/excluded services, and hotel ratings for 12 countries
- **Package booking** — dropdown selector to pick a package and start a booking
- **Custom tour form** — a dedicated `form.html` page for requesting a custom tour
- **About, Why Choose Us, and Contact sections** — company info, key selling points, and contact details
- **Fully responsive layout** with a clean nav, hero section, and footer with social links

## 🛠️ Built With

- **HTML** — page structure and content
- **CSS** — styling and layout (`css/style.css`)

## 🚀 Getting Started

No build tools or dependencies required — it's a static site.

1. Clone the repo
   ```bash
   git clone https://github.com/Haziq8900/azure-horizons-travel-agency-website.git
   ```
2. Open `index.html` in your browser

That's it — you're exploring Azure Horizon.
